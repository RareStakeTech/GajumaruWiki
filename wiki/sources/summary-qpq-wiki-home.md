## Documentation Principles

| Principle | Meaning |
|-----------|---------|
| JUST WRITE | Imperfect content > nonexistent content |
| PICTURES | Every article should have diagrams |
| QUICKREF | Practical getting-started sections with examples |
| DEMOS | Functional code examples throughout |
| CENTRALIZE | All technical docs on the wiki |
| LINK | Source code references and receipts |
| VIDEO | Complex topics have video explanations |
